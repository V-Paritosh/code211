Palatine Website Workshops

@ 10:15AM

Beginner: Static websites

Host: Jacob Kim and Tanish Mittal

This beginner-level session comprehensively introduces building static websites using fundamental HTML concepts. Participants will learn the basics of structuring web pages, incorporating JavaScript for interactivity, and understanding essential HTML elements. By the end of this workshop, attendees will have a solid foundation to create their own simple static web pages.

Software Needed:
  • VS Code

Advance: Dynamic Websites

Host: Isaac Ruiz and Norbert Zych

In this advanced workshop, participants will advance their skills in web development by learning to build dynamic websites using JavaScript. The session will cover topics such as event handling, asynchronous programming, and integrating dynamic content. Through hands-on projects, attendees will gain proficiency in creating interactive web applications capable of responding to user input and data changes in real time.

Software Needed:
  • VS Code
  • Node.js

Fremd ML & AI Workshops

@ 11:15AM

Beginner: Introduction to Machine Learning using Kaggle

Host: Jason Zhou

Learn the basics of Machine Learning and what it’s all about using Python and a step-by-step tutorial to create a model that predicts which passengers survived the wreck of the Titanic. Attendees will learn how to use basic modules to analyze tabular data.

Software Needed:
  • Kaggle

Advance: Use computer vision to draw like an artist

Host: Ryan Barretto

Build generative adversarial networks (GANs) from scratch to mimic the artistic styles of Picasso, Van Gogh, and others to create beautiful, never-seen-before images. You will learn how to build complex neural networks from scratch and train them using sophisticated optimization strategies.

Software Needed:
  • Private google account
  • Link to Kaggle

Schaumburg API's Workshop

@ 2:00PM

API’s & Postman

Host: Paritosh Vaghasiya, Ishaan Patel, Vedant Patil, and Ibraim Kanza

Join us for an immersive workshop on APIs (Application Programming Interfaces), designed to equip you with the skills needed to become proficient in understanding API development. Delve into the fundamentals of APIs, including endpoints, requests, responses, and parameters, gaining a solid understanding of their structure and functionality. Learn about API usage across different industries, from social media integration to e-commerce platforms, to grasp the versatility of APIs in modern applications. Through hands-on examples, learn how to access and modify information using APIs, gaining valuable experience in working with external data sources. While this workshop serves as a starting point, it will provide you with a solid introduction to APIs and ignite your curiosity for further exploration in your development journey.

Software Needed:
  • Any type of computer
  • A personal email that you’ll have access to on Hackathon day
